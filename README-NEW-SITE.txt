BEAUTY OF JOSEON IRAQ — إعداد النسخة الجديدة

1) افتح Supabase وأنشئ مشروعاً جديداً.
2) افتح SQL Editor وشغّل ملف setup-supabase.sql كاملاً.
3) افتح Storage وتأكد أن bucket باسم product-images موجود وعام Public.
4) افتح Authentication > Users وأنشئ مستخدم الإدارة بالبريد وكلمة السر.
5) من Project Settings > API انسخ:
   - Project URL
   - Publishable key (أو anon public key)
6) افتح supabase-config.js واستبدل القيمتين فقط.
7) عدّل رقم واتساب من لوحة الإدارة بعد تسجيل الدخول.
8) ارفع جميع الملفات كما هي إلى GitHub Pages.

مهم: النسخة لا تحتوي منتجات تجريبية، ولا يوجد حد 100 قطعة. الحد الوحيد هو كمية المخزون المسجلة للمنتج.
