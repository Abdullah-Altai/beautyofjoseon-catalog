موقع كتالوك Beauty of Joseon Iraq
=================================

الموقع صار كتالوك فقط:
- خلفية بيضاء وتصميم قريب من كتالوك PDF.
- المنتجات صغيرة ومجمعة حسب كل قسم في صفحة واحدة.
- لا توجد سلة ولا أزرار إضافة أو طلبات.
- الضغط على المنتج يفتح الصور والتفاصيل فقط.
- لوحة الأدمن تتحكم بالمنتجات، الأقسام، الإعلانات، الغلاف، النصوص، الألوان والترتيب.
- حساب الأدمن الوحيد: Altai0193@gmail.com

الربط مع Supabase
-----------------
1. أنشئ مشروعاً جديداً في Supabase.
2. افتح SQL Editor وشغّل ملف setup-supabase.sql بالكامل مرة واحدة.
3. افتح Authentication ثم Users ثم Add user.
4. أنشئ المستخدم بالبريد: Altai0193@gmail.com
5. اختر كلمة مرور وفعّل Auto Confirm User.
6. من Project Settings ثم API انسخ:
   - Project URL
   - Publishable Key أو anon public key
7. افتح supabase-config.js وضع القيمتين مكان النصوص الجاهزة.

مهم
----
Supabase يحفظ قاعدة البيانات والصور وتسجيل دخول الأدمن، لكنه ليس استضافة عادية
لملفات HTML. بعد الربط ارفع مجلد الموقع إلى GitHub Pages أو Netlify أو Vercel.

الصفحات
-------
- index.html       الكتالوك العام
- admin.html       لوحة الإدارة

لا تضع Service Role Key داخل ملفات الموقع. استخدم Publishable Key فقط.
